## Comments
- Part of code which is not executed
```js
// This is a single line comment
/* This is a muti-line comment */ 
```

## Operators
- Used to perform some operation on data

## Binary Operators
1. Arithmetic Operators (+, - , *, /)
```js
let a = 5;
let b = 2;

console.log(a+b); // 7
console.log(a-b); // 3
console.log(a*b); // 10
console.log(a/b); // 2.5
```
2. Modulus
```js
console.log(a%b); // 1
```
3. Exponentiation
```js
console.log(a**b); // 5^2 = 25
```
## Unary Operators
1. Pre Increment
```js
console.log(++a); // 5->6
```
2. Post Increment
```js
console.log(a++); // 5->5
```
3. Pre Decrement
```js
console.log(--b); // 2->1
```
4. Post Decrement
```js
console.log(b--); // 2->2
```
## Assignment operators
- An assignment operator assigns a value to its left operand based on the value of its right operand.
